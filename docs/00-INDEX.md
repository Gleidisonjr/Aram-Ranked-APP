# Index do projeto (docs por feature)

Este repositório tem documentação separada por intenção/feature para facilitar manutenção e para você enxergar “o que muda” sem procurar em um único arquivo gigante.

## Como usar
1. Primeiro veja o **modo admin**.
2. Depois veja como funciona a **sincronização multi-admin** (para atualizar `public/ranking.json`).
3. Por último, veja como fazer **deploy do GitHub Pages**.
